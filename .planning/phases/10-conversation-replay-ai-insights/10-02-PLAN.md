**1. Create `src/lib/admin/insights-prompt.ts`** (Claude's discretion on exact prompt, per D-12):

- Export `INSIGHTS_SYSTEM_PROMPT` constant — a string in Portuguese instructing Claude to analyze a consortium sales conversation and extract structured insights
- The prompt MUST instruct:
  - Respond with ONLY valid JSON, no markdown, no explanation, no code fences
  - Extract exactly 4 fields: `intent`, `budget`, `objections`, `next_action`
  - Format:
    ```json
    {
      "intent": "string - o que o cliente quer (tipo de bem, prazo, contexto)",
      "budget": { "monthly": number|null, "total": number|null, "notes": "string" },
      "objections": ["string"] or [],
      "next_action": "string - acao recomendada para o vendedor"
    }
    ```
  - If information is not available, use null for numbers and empty strings/arrays as appropriate
  - Focus on actionable insights, not just summarizing
- Export `buildInsightPrompt(messages: Array<{ role: string; content: string }>)` function:
  - Takes the conversation messages array
  - Returns a string with the conversation formatted as a readable transcript:
    ```
    [Cliente]: mensagem...
    [Agente]: resposta...
    ```
  - Prefix with: "Analise a seguinte conversa entre um cliente e um agente de consorcio IA:"

**2. Create `src/app/api/admin/leads/[id]/insights/route.ts`** (per D-15):

- Export `POST` handler with signature: `(req: Request, { params }: { params: Promise<{ id: string }> })`
- Call `requireRole("admin", "viewer")` — return error if unauthorized
- Extract `leadId` from params. Validate UUID format. Return 400 if invalid.

**Cache check (per D-13, D-14):**
- Query `lead_insights` table for this leadId: `db.query.leadInsights.findMany({ where: eq(leadInsights.leadId, leadId) })`
- If cached entries exist AND the most recent `generatedAt` is less than 1 hour old: return cached insights immediately as `{ insights: cached, source: "cache" }`
- Format cached response: group by `insightType`, return `{ intent, budget, objections, next_action }` parsed from the `content` text field

**Fetch conversation:**
- Use same Drizzle relational query as conversation route to get messages:
  ```typescript
  const lead = await db.query.leads.findFirst({
    where: eq(leads.id, leadId),
    with: {
      conversation: {
        with: {
          messages: {
            orderBy: (messages, { asc }) => [asc(messages.createdAt)],
          },
        },
      },
    },
  });
  ```
- If no lead or no conversation or 0 messages: return 400 `{ error: "Nenhuma mensagem para analisar" }`

**Generate insights (per D-16):**
- Import `generateText` from `"ai"` and `createAnthropic` from `"@ai-sdk/anthropic"`
- Call:
  ```typescript
  const anthropic = createAnthropic();
  const { text } = await generateText({
    model: anthropic("claude-haiku-3-5-latest"),
    system: INSIGHTS_SYSTEM_PROMPT,
    prompt: buildInsightPrompt(lead.conversation.messages),
  });
  ```
- Parse the response: strip any accidental markdown fences (```json ... ```) before `JSON.parse`. Wrap in try/catch — if parse fails, return 500 `{ error: "Falha ao processar insights" }`

**Persist to DB (per D-13):**
- Delete existing insights for this lead: `db.delete(leadInsights).where(eq(leadInsights.leadId, leadId))`
- Insert 4 rows (one per insight type): intent, budget (JSON.stringify the budget object), objections (JSON.stringify the array), next_action
- Each row: `{ leadId, insightType, content: JSON.stringify(value), model: "claude-haiku-3-5-latest", generatedAt: new Date() }`
- Use `db.insert(leadInsights).values([...])` for batch insert

**Return response:**
- Return 200 with `{ insights: parsedInsights, source: "generated" }`
- The `insights` object has shape: `{ intent: string, budget: { monthly, total, notes }, objections: string[], next_action: string }`

| T-10-06 | Denial of Service | Repeated insight regeneration | mitigate | 1hr cache TTL prevents redundant API calls. Could add rate limiting in future. |
