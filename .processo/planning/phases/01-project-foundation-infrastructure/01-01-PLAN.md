### Task 3: Configure Biome with project standards

<read_first>
- biome.json
</read_first>

<action>
Do not change the `$schema` URL generated by create-next-app. Only ensure `formatter.indentStyle` is `"tab"` and `formatter.lineWidth` is `100`. Update those keys if they differ from the defaults. Leave all other keys (organizeImports, linter, etc.) as scaffolded.

Add npm scripts to `package.json` if not already present:
- `"lint": "biome check ."`
- `"format": "biome check --write ."`

Run `npx biome check --write .` to fix any formatting issues in the scaffolded code.
Then run `npx biome check .` and confirm exit code 0.
</action>

<acceptance_criteria>
- `biome.json` contains `"indentStyle": "tab"`
- `biome.json` contains `"lineWidth": 100`
- `package.json` contains `"lint"` script
- `package.json` contains `"format"` script
- `npx biome check .` exits with code 0
</acceptance_criteria>
