## Cenário

- **Rota/tela:** não é um bug de UI — é regressão de comportamento do agente
  (gates de reveal + copy do system prompt), capturada via suíte de regressão.
- **Passos pra reproduzir:** na branch `develop` (HEAD atual, sem nenhuma mudança),
  rodar:
  ```
  pnpm vitest run \
    src/lib/agent/qualify-state.fix-275-motivo-nao-trava.test.ts \
    src/lib/agent/qualify-state.fix-296-reordena-funil.test.ts \
    src/lib/agent/system-prompt.fix53.test.ts \
    src/lib/whatsapp/no-emoji-fix212.test.ts
  ```
  As 4 falhas reproduzem, ou rode a suíte inteira (`pnpm test:unit`) — falha nesses
  mesmos 4 arquivos.
- **Confirmação de que é debt da base:** `git diff origin/develop...HEAD -- src/lib/agent src/lib/whatsapp`
  não mostra nenhuma diferença nesta branch (`feat/kv-topo-conversao`) — ou seja,
  as falhas já existem tal como estão na `develop`, não foram introduzidas por
  este bloco. Confirmado também isolando as mudanças do bloco via `git stash`
  (roda igual, falha igual, com ou sem o stash).
- **Blast radius:** bloqueia o pre-commit hook (husky, gate `pnpm test:unit`) de
  QUALQUER branch/worktree forkada da `develop` atual — não só deste bloco. Todo
  bloco paralelo desta mesma onda (`bloco-b-kv-narrativa-jornada`,
  `bloco-c-kv-confianca-fechamento`) provavelmente bate no mesmo gate ao commitar.

## Esperado × Atual

     mudou de redação sem manter o marcador que o teste procura).

   exemplo (dentro de aspas)".
